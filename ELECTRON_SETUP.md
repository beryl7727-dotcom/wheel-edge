# Wheel Edge - Electron Desktop App Setup

Your Wheel Edge Trading Dashboard is ready to run as a desktop application (Windows, macOS, Linux).

## ⚡ Quick Start (Desktop App)

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2a: Run in Development Mode (with hot reload)
```bash
npm run electron-dev
```

This will:
- Start the React dev server
- Open Wheel Edge in an Electron window
- Auto-reload when you edit code
- Show DevTools for debugging

**Exit:** Close the Electron window or press Ctrl+C in terminal

### Step 2b: Build for Production

#### Windows:
```bash
npm run electron-build
```
Creates: `dist/Wheel Edge.exe` (installer) and portable .exe

#### macOS:
```bash
npm run electron-build
```
Creates: `dist/Wheel Edge.dmg` (installer)

#### Linux:
```bash
npm run electron-build
```
Creates: `dist/Wheel Edge.AppImage` (executable)

#### All Platforms (requires build tools):
```bash
npm run electron-build-all
```

Your executables are in the `dist/` folder:

### Windows

- **Wheel Edge Portable.exe** → No install, just run

## 🎨 Customizing Your App

### App Icon
Add icons to `public/`:
- `icon.png` (1024x1024) - used for all platforms
- `icon.ico` - Windows specific
- `icon.icns` - macOS specific

Icons are automatically included in builds.

### App Name & Version
Edit `package.json`:
```json
{
  "name": "wheel-edge-trading-dashboard",
  "version": "0.1.0",
  "build": {
    "appId": "com.wheeledge.trading",
    "productName": "Wheel Edge"
  }
}
```

### Window Size
Edit `main.js`:
```javascript
mainWindow = new BrowserWindow({
  width: 1920,        // Default width
  height: 1080,       // Default height
  minWidth: 1280,     // Minimum width
  minHeight: 720,     // Minimum height
});
```

### Menu Items
Edit `main.js` menu template:
```javascript
const template = [
  {
    label: 'File',
    submenu: [
      { label: 'Exit', accelerator: 'CmdOrCtrl+Q', click: () => app.quit() }
    ]
  },
  // Add more menus here
];
```

## 🐛 Troubleshooting

### "npm: command not found"
Install Node.js from https://nodejs.org

### Electron window won't open
```bash
# Kill any running processes
killall node        # macOS/Linux
taskkill /F /IM node.exe  # Windows

# Try again
npm run electron-dev
```

### DevTools not opening
Edit `main.js` line ~30:
```javascript
// Set to true to see DevTools
if (isDev) {
  mainWindow.webContents.openDevTools();
}
```

### Build fails
```bash
# Clean and reinstall
rm -rf node_modules dist build
npm install
npm run electron-build
```

### IndexedDB not persisting
IndexedDB data is stored in:
- **Windows:** `%APPDATA%/Wheel Edge/`
- **macOS:** `~/Library/Application Support/Wheel Edge/`
- **Linux:** `~/.config/Wheel Edge/`

Delete folder to reset app data.
